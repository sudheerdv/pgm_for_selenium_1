Create a Repository
------------------------------------------
1. login in to gitHub.com
2. click'New' or click '+' -> New Repository
3.specify 'Repository name'  RF25
4. Click Create Repository
5.Copy the Repository URL (URI)
https://github.com/javabhanu/RF25.gitHub


Uploading JavaProject (framework) to github
-------------------------------------------
1. Right Click on JP -> team->Share Project
2. Click create; type '\RF25' after the path,click finish
again click Finish.
3. Right Click on JP -> team->commit
4. drag and drop all the files from 'unstaged' to 'staged'
5. Specify the message , click 'Commit & push'
6. Specify 'URI' , UN & PED , select the checkbox,Click Next
7.Click Next
8.Click Finish
9. Click Close() (if there any popup for password hint click NO)

Importing Framework(JavaProject) from GitHub to Local System
----------------------------------------------
1. In Eclipse go to file import
2. Expand Git -> Select Projects from Git->next
3. select clone URI -> next
4. Specify URI  username and password  , Select the checkbox -> click next
5.click next(if there is any error change the name of the folder RF25a)

Ensure that first radio button is selected -. click next -> click Finish
